Bienvenidos a Soldty

“Soldty” es un aplicativo compatible con dispositivos Android 4.1+

  • Para su correcto uso, se recomienda conexion constante a Internet
  • Su dispositivo debe tener activo el servicio de GPS
  • Permite guardar favoritos y revisarlos sin conexion
  • El mapa interactivo brinda la ubicacion exacta de la propiedad

Mapa

El mapa interactivo de “Soldty” utiliza los servicios actualizados de Google Maps para brindar una correcta ubicacion de la propiedad.

El mismo permite el uso de “multi touch” para moverse comodamente por el mapa y realizar zoom con mayor comodidad.

Botones en pantalla le permitiran realizar el recorrido desde su ubicacion actual hasta la propiedad que se seleccione en pantalla.

Propiedades

La lista principal le dara una informacion concisa de las propiedades y al tocar sobre la misma, abrira una pantalla con detalles completos para una informacion detallada y precisa.

El boton “+” es un menu dedicado para realizar acciones particulares:

  • La “estrella” permite guardar una propiedad como favorita para su lectura “online/offline”
  • El buzon permite contactar via “Email” con el agente o particular de dicha propiedad
  • El icono de telefono permite una comunicacion directa con el representante de la propiedad

El boton “VER MAS” permite acceder a todas las propiedades publicadas por dicho agente o particular.

Esta opcion requiere conexion a Internet y no se puede acceder “offline”.

De contar con una Galeria, la misma sera mostrada por debajo de la descripcion de la propiedad

Por ultimo, se cuenta nuevamente con un mapa interactivo para conocer la ubicacion exacta.

Ajustes

Para acceder a los ajustes, debe tocar sobre los tres puntos en el margen superior derecho.

Esta opcion le permite conocer el cambio oficial del Dolar en tiempo real.

Una opcion permite cambiar entre la moneda local o el Dolar Americano.

La opcion de “Avisos Premium” permite un filtrado rapido en la pantalla principal para visualizar avisos destacados.

La version del aplicativo se detalla en el margen inferior para asegurarse de estar en la version mas actualizada y con las ultimas caracterasticas.

Acceso Agentes

De tener creada una cuenta en nuestro sitio web de "Soldty", podra hacer uso de su cuenta y credenciales para acceder al aplicativo y revisar sus avisos, modificar datos personales en pantalla y nuevas funcionalidades que se iran incorporando en nuevas actualizaciones.

Busqueda

En el margen superior derecho cuenta con un acceso rapido a busquedas filtradas.

Un campo editable le permitira completar su busqueda para filtrar las propiedades que mas le interesan y/o desea averiguar.

Esta funcionalidad ira incorporando mas caracteristicas de filtrado para mejorar su experiencia y obtener los resultados precisos.
